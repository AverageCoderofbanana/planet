# CSV Data Analysis Project

A data analysis project focused on extracting meaningful insights from structured CSV datasets, combined with basic machine learning and AI-powered interaction.

## Overview

This project explores how raw data can be cleaned, analyzed, and transformed into insights using Python. It also integrates a lightweight machine learning model for future predictions and a Gemini-based chatbot for interactive data understanding.

## Features

* Data cleaning and preprocessing using pandas
* Pattern analysis and insight extraction
* Data visualization using matplotlib and seaborn
* Climate prediction model (2040 forecast) using linear regression
* Gemini-powered chatbot for interacting with the dataset

## Tech Stack

* Python
* pandas, numpy
* matplotlib, seaborn
* scikit-learn (Linear Regression)
* Google Gemini API (Gemini 2.5 Flash)
* requests, re

## How It Works

* Load and clean CSV dataset using pandas
* Perform analysis to identify trends and patterns
* Visualize data using graphs for better understanding
* Train a linear regression model to predict future climate trends (2040)
* Integrate Gemini API to allow user interaction with the dataset through a chatbot
